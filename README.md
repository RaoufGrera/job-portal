# Jobs portal
Job portal using pure php with mysql database
## Docker
`docker-compose up --build -d`

## Features
- Resume builder
- Search engine
    - seekers
    - jobs
- Create company profile
- Post jobs
- And more

## Built With
* [PHP](https://www.php.net/)
* [MySQL](https://www.mysql.com/)
* [JQuery](https://jquery.com)
* [Bootstrap](https://getbootstrap.com)
